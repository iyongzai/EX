# EX
EX是一个对系统类进行扩展的工具开源库，针对String，Array，Double，UIImage，UIView等类扩展了一些常用的方法，使用方便

## Installation and Setup

*最新版本支持Swift 5*

AYSeg works on Xcode 7-11 and above, iOS 9-13 and above.

## Requirements

* IOS 9.0+
* Xcode 10+
* swift 4+

### CocoaPods

Podfile文件配置：

Swift 4.0  --> pod 'AYSeg', '0.0.7'

Swift 5.0 --> pod 'AYSeg', '0.5.0'
